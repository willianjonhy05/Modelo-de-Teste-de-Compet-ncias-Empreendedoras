# Teste de Autoavaliação de Competências Empreendedoras

Este projeto é uma aplicação web simples para autoavaliação de competências empreendedoras, baseada no modelo de **McClelland (1987), adaptado**.

A proposta do sistema é permitir que o usuário responda a um questionário com 39 afirmações e, ao final, visualize sua pontuação em 13 características empreendedoras.

## Objetivo

O objetivo deste projeto é oferecer uma ferramenta prática, visual e de fácil utilização para apoiar processos de autoanálise comportamental no contexto do empreendedorismo.

## Funcionalidades

- Questionário com 39 perguntas
- Respostas por escala de 1 a 4
- Cálculo automático das pontuações
- Armazenamento temporário dos dados com `localStorage`
- Página de resultado com tabela organizada
- Interface limpa, responsiva e de fácil leitura

## Características avaliadas

O sistema apresenta resultados para as seguintes competências:

- Estabelecimento de metas
- Riscos calculados
- Qualidade
- Planejamento
- Busca de Informações
- Busca de oportunidades
- Iniciativa própria
- Persistência
- Eficiência
- Persuasão
- Necessidade de independência
- Autoconfiança
- Comprometimento
