# Modelo-de-Teste-de-Compet-ncias-Empreendedoras
Aplicação web de autoavaliação empreendedora baseada no modelo de competências comportamentais de McClelland (1987), adaptado, desenvolvida com HTML, CSS e JavaScript. O sistema permite responder a 39 questões, processa automaticamente os resultados e apresenta a pontuação final em 13 características empreendedoras.
